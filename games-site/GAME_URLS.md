## Testing URLs from School

Before adding a URL:
1. Test it works from school network
2. Try both desktop and mobile
3. Check if it loads in an iframe (if not, opens in new window)
4. Verify no authentication required

## Adding Games in Admin Panel

1. Login as admin
2. Go to Admin tab
3. Fill in:
   - **Name**: Game title (e.g., "2048")
   - **URL**: Full URL (e.g., "https://play2048.co/")
   - **Image URL**: Cover image URL
   - **Type**: Select "Game" or "App"
4. Click "Create"
5. Game appears in Games/Apps tab immediately

## Frequently Asked Questions

**Q: Why doesn't this game work?**
A: School network might block it, or it requires Flash/Unity (outdated). Try a different game.

**Q: Can I use YouTube?**
A: Yes! Many schools allow YouTube. Add https://youtube.com/ as an app.

**Q: How do I make games work if they're blocked?**
A: Use proxy services or games hosted on Google's servers (withgoogle.com).

**Q: What if a game has authentication?**
A: Game won't work - choose games that don't require login.

**Q: Can I add paid games?**
A: Only free games. Paid services require credentials.
